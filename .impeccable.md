# Passo Animation Studio — Design Context

## Design Context

### Project Overview

- **Product**: パッソアニメーションスタジオ LP / 株式会社Passo a Passo コーポレートサイト
- **Domain**: passo-ww.com (愛知県岡崎市・豊田市)
- **Service**: 就労継続支援B型 — 企業から映像・イラスト制作を受託するクリエイターチーム
- **Work mix**: 外部クライアント案件と自社メディア運営の両方を実制作
  - 自社メディア: TikTok「ココロカルテ」等、フォロワー1万人規模（本部経由の社内発注扱い）
  - 公開コピーでは「フォロワー1万人規模の自社メディア」として差別化に活用可
- **Primary goal**: Instagram/Google Display広告からの流入で「ここに通いたい」と思わせ、LINE問い合わせに誘導
- **Conversion**: 見学→登録転換率ほぼ100%。ボトルネックは見学数のみ。
- **Tech**: Next.js 16, React 19, App Router, plain CSS (no Tailwind), Japanese

### Users

**Primary reader**: 20代女性、精神疾患（うつ・双極性障害・統合失調症等）または発達障害（ASD・ADHD等）、現在無職・引きこもり気味のクリエイター志望

**Profile details**:
- 生活基盤は親が支えており、経済的切迫感がない（お小遣い程度の収入でもOKな層）
- 趣味・副業で動画編集を続けてきたが、クラウドソーシングで壁にぶつかっている
- 「ちゃんとした仕事がしたい」という自己認識とプライドがある
- でじるみ・にじげん等（同業他社）には「自分には合わない」と感じている
- 自己評価は低めだが、プライドはある
- スマートフォンがメイン。Instagram日常利用。自宅閲覧。
- 0.5秒で広告を判断してスクロールを止めるかどうか決める

**Three visitor patterns** (優先度順):

| パターン | 状態 | 集客難易度 | 主な訴求 |
|---------|------|-----------|---------|
| A | 既存B型利用者で「ここじゃない」と感じ次を探している（受給者証あり） | 低 | 「企業案件がある」一言で反応 |
| B | でじるみ等を調べたが未利用（ある程度知識あり） | 中 | 「ここは違う」という差別化 |
| C | 就労継続支援B型を知らない最大母数層 | 高 | 「企業案件の映像制作チーム」として福祉色なく引き込む |

**Secondary reader**: 20〜30代男性、同様の背景・価値観（大歓迎）

**Emotional goal**: 「ここに通いたい」→LINEで問い合わせに進む状態になること

### Brand Personality

**3 words**: 誠実 · おしゃれ · 仲間（Sincere · Aspirationally Stylish · Welcoming as peers）

**Extended tone**:
- 「ここに所属すること自体が自尊心を満たす」感覚を与える
- スタバでMacBookを開くノマドスタイルへの憧れに近いおしゃれさ
- 「来るだけでワンランク上の自分になれる」感覚
- 嘘がない、飾らない温かさ
- 上から目線・「支援してあげます」感ゼロ
- 対等な語りかけ。同じ目線。

**Core value proposition** (LPで伝えるべき3つ):
1. **実務経験が積める** — 練習・学習ではなく、実際の制作業務（企業案件・自社メディア）に関わる
2. **ポートフォリオが積み上がる** — 実際の制作に関わるたびに、実績として残っていく
3. **営業・工程管理・クライアント対応を体感できる** — スクールでは学べないこと

**Competitors to differentiate from** (their feel = our anti-reference):
- でじるみ・にじげん → 「オタクのたまり場」「お遊びみたいなスクール」「可愛らしい居場所型」
- ギラギラした福祉サイト（重いビジュアル、「応援してます」感）
- 量産型コーポレートサイト（冷たい、無機質）

### Aesthetic Direction

**Visual tone**: 洗練された小規模制作スタジオ — 福祉施設でもスクールでも企業オフィスでもない。「プロが集まって仕事をする場所」であり、かつ温かく迎え入れてくれる。

**The core aesthetic tension**: Japandi の穏やかな落ち着きを保ちながら、「ここは本物の仕事の場所だ」というスタジオ感・信頼感を加える。

**Motion**: スクロールリビールはエレガントに、穏やかに。バウンス・攻撃的なエントランスNG。`prefers-reduced-motion` 必須対応。

---

### Design Principles

1. **仕事感を前面に** — 企業案件・実務経験・ポートフォリオを軸として語る。居場所・スクール・支援施設に見えてはならない。ビジュアルもコピーも「制作スタジオ」として一貫させる。

2. **自尊心を守る美しさ** — 「ここに所属する自分が誇らしい」と感じさせる。デザインが整っていること自体が「自分はこういう場所にいていい」という自己肯定につながる。圧迫ゼロ。

3. **情報量の制御** — 精神疾患・発達障害のある方が読む前提。一画面に詰めすぎない。余白（間）を守る。認知負荷を最小化する。

4. **対等な語りかけ** — 禁止ワードリストを厳守。上から目線・プレッシャー・自己否定を生む言葉は徹底排除。

5. **差別化の精度** — でじるみ・にじげんのLPが「来てほしくない層を集めてしまう設計」。パッソのLPは「企業案件に関わりたいプライドある層のみ」に刺さる設計。ターゲット外が「自分には合わない」と離脱するのは正解。

### UX Writing Rules

**禁止ワード（厳守）**:

| カテゴリ | 禁止語 | 理由 |
|--------|--------|------|
| 自己否定系 | プロ・本物の・本格的・本気で・武器・選ばれるクリエイター・実務レベル（広告内）| 「今の自分はまだ〇〇ではない」という否定感 |
| プレッシャー系 | 消耗・戦う・勝つ・挑戦・もしそうなら・対象です | 緊張・プレッシャーを生む |
| 福祉施設系 | 居場所・自分らしく・自分のペースで・ゆっくり・無理なく・支援（単独）・お気軽に・基礎から教えます | 福祉色・スクール色 |
| スクール系 | スキルを磨く・学べます・身につきます・体験できます | スクールのイメージ |
| 好き系 | 「好きを仕事に」系全般 | でじるみ・にじげんとの差別化 |
| 読みにくい語 | 携わる | 画数多・難解 → 「関わる」に変更 |
| 文脈違い | 「作品」（日常業務・LP・広告文脈） | アート/趣味/居場所系の響き → 「制作物」「納品物」に置換。「作品」は年1回展覧会文脈でのみ使用可 |

**文脈によって使用可能**:
- `ポートフォリオ` — LP内OK、広告内NG
- `選ばれるクリエイター` — LP内検討中、広告内NG
- `実務レベル` — LP内検討中、広告内NG
- `在宅支援` — 「行っておりません」という事実文脈のみ可

**推奨表現**:
- 企業案件に関わる / 映像制作チーム / ポートフォリオが積み上がる
- 実績が残る / チームで制作する / 企業から受託した仕事
- 企業案件と自社メディアの実制作 / フォロワー1万人規模の自社メディア
- 制作スタジオ / 工程を分担 / 納品まで関わる
- 制作物 / 納品物 / クライアントの制作物 / 納品物の品質

**用語の使い分け（社内ルール / 厳守）**:

| 文脈 | 呼称 | 位置づけ | 目的 |
|---|---|---|---|
| 日常業務・LP・広告 | **制作物 / 納品物** | ビジネス（商業クリエイターとしての活動） | クライアント要望に応えるスキル習得、自立に向けた実務経験 |
| 年1回の展覧会（イベント） | **作品** | アート（アーティストとしての活動） | 個人の純粋な表現、創造性の発揮、ハレの舞台 |

**CRITICAL**: LP・広告・日常業務の文脈で「作品」を使うと、アート/趣味/居場所系の響きになり差別化が崩れる。必ず「制作物」「納品物」を使う。「作品」は年1回展覧会に限定。

### Accessibility Requirements

- **Contrast**: WCAG AA以上（本文・小文字含む）
- **Font size**: 本文15px以上（モバイル）
- **Information density**: 一画面に詰めすぎない。スクロールで少しずつ開示する。
- **Motion**: `@media (prefers-reduced-motion: reduce)` 必須対応（既存実装あり）
- **Tone**: 「頑張れ」「できますよ」系の圧迫感ゼロ（コピーだけでなくビジュアルトーンも含む）
- **Kanji**: 複雑な漢字を避け、読みやすいひらがな・やさしい漢字のバランスを保つ
