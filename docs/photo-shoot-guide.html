01

撮影の方針

パッソが伝えたいのは「福祉施設」ではなく「プロの制作スタジオ」です。
20代のクリエイター志望者が「ここに通いたい」と思う、おしゃれで先進的な写真を目指します。

写真で表現したいこと
  • 制作物・編集画面が主役（人物は脇役）
  • 少し暗めのシネマティックなトーン
  • 背景や手前を大胆にボケさせる「奥行き」
  • 「指導されている」ではなく「自律して作業している」構図
避けるトーン

福祉施設の事務室を伝えるような証拠写真／笑顔で正面を向くポーズ／全員カメラ目線／蛍光灯のベタ光／凡庸な広告写真。

02

iPhone 16 Pro の設定

1.「ポートレート」モードを選ぶ

カメラアプリを開いて、下のメニューから「ポートレート」を選択します。
人物のまわりが自動的にボケて、奥行きのある写真になります。

2.「f / 値」を 1.4 〜 2.0 に設定

画面右上の「f」マークをタップして、数字を 1.4 〜 2.0（小さい数字）に設定します。
数字が小さいほどボケが強くなり、シネマティックな雰囲気になります。

3.少し暗めに撮る

画面のどこかをタップした後、太陽マーク（明るさ調整）を 少し下にスライド。
明るすぎると安っぽく見えるので、ほんの少しだけ暗めが正解です。

4.縦・横どちらでもOK

LPでは横位置・縦位置どちらも使います。同じ被写体で 両方撮影しておくと安心です。

03

撮影で守ってほしい3つのルール

RULE 01
「人」じゃなくて「画面」にピントを合わせる

撮影前に、iPhone画面のなかで モニターをタップ。ピントを画面に固定します。
手前の髪・肩・手はボケてOK。むしろ大胆にボケさせる方が「プロっぽい」写真になります。 主役は制作物・編集画面であって、人ではありません。

RULE 02
少し暗めに、シネマティックに

ピントを合わせた後、画面の 太陽マーク を少し下にスライドして暗めに。
蛍光灯のベタ光だけだと安っぽくなるので、可能なら ブラインドを少し開けて 窓からの光も取り入れる。光と影のコントラストが「映画みたいな空気感」を作ります。

RULE 03
被写体を画面の真ん中に置かない

人物やパソコンを画面の 少し左寄り or 右寄りに配置。
余白を残すことで、雑誌の誌面のような「編集された写真」になります。中央揃えは広告写真っぽくなり、おしゃれ感が落ちます。

04

撮影前のチェックリスト（5分）

  • 机の上の 私物を片付ける（カップ・お菓子・付箋・書類・ティッシュ）
  • パソコン画面に Premiere Pro や Photoshop / Illustrator を開いておく（できれば実案件のタイムライン）
  • 画面の輝度は 最大近く に上げる（暗いと写真上で潰れる）
  • ブラインドを 少し開ける（窓からの自然光を入れる）
  • 映る人の服は 無地・落ち着いた色（蛍光色・大きな柄・ロゴはNG）
  • マスクの色も 白か黒を推奨（不織布の青・ピンクは避ける）
  • 顔出しNGの方が映り込まないか 事前確認
05

やる・やらない

DO ✓
  • 編集画面にピントを合わせる
  • 少し暗めの露出
  • 窓からの自然光を入れる
  • 被写体を画面の端に寄せる
  • 後ろ姿・横顔・手元のショット
  • 同じシーンで複数枚撮る
DON'T ✕
  • カメラ目線・笑顔のポーズ
  • iMacを真正面から撮る
  • 蛍光灯のみのフラットな光
  • 全員が画面に注目する演出
  • 派手な色・柄の服が映る
  • 「指導している」上下関係の構図
06

必要なカット（ショットリスト）

各事業所で以下のカットを撮影してください。各カット 5〜10枚ずつ撮ると後で選びやすいです。

01 スタジオの俯瞰（広角・無人） Wide / Empty
  • 誰もいない、整った状態のスタジオ全景
  • パーテーションとiMacが整列して見える構図
  • 自然光とブラインドの陰影あり
  • 朝の早い時間 or 昼休み中の静かな時間がベスト
02 編集中の手元（接写・ボケ強） Close-up / Hands
  • Premiere Pro のタイムライン部分にピントを合わせる
  • キーボードに置かれた手はボケで動きを表現
  • 横や斜め後ろから狙う
03 2人での画面相談（横位置） Collaboration
  • モニター画面の指している先にピント
  • 2人の顔は半分ボケる（後ろ姿・横顔）
  • 「指導されている」ではなく「対等に相談している」構図
  • 後ろから2人を撮る角度がおすすめ
04 ペンタブで描画中（手元中心） Illustration
  • ペンタブのペン先・キャラクター描画中の画面にピント
  • 髪・服はボケる
  • モニター内の絵に集中させる構図
05 チーム全体の俯瞰（複数人作業中） Team
  • パーテーション越しに複数の制作風景
  • 後ろから・斜め上から（脚立や椅子の上から狙う）
  • 全員の作業画面が少しずつ見える
06 環境ディテール（小物寄り） Detail / Object
  • iMac、ペンタブ、キーボード、植物、観葉植物
  • 物の質感を活かした静物的なカット
  • 「スタジオの世界観」を演出する補完カット
07

撮影後の流れ

  1. 撮影した写真をすべて、Teams の「アニメ共通 / 01_業務連絡」にアップロード
  2. 各カット 5〜10枚ずつ送ってください（こちらで選定します）
  3. 加工は不要。撮りっぱなしのオリジナルデータで送ってください
  4. 窪田さんが採用カットを選定し、必要に応じて色補正してLPに反映
送り方のヒント

iPhoneから直接Teamsへアップロードする際、「実際のサイズ」を選んでください。
圧縮されたサイズだと、後でLPに使うときに画質が足りなくなります。
